# SecureLog Watcher

## 🔍 프로젝트 소개
SecureLog Watcher는 보안 로그를 수집·분석하고, 이상 징후를 시각화하여 보여주는 **경량 SIEM PoC 프로젝트**입니다.

Spring Boot + React 기반으로 설계되었으며, SIEM의 핵심 원리를 직접 구현하는 것을 목표로 합니다.

## 🎯 프로젝트 목적
- 로그 수집 → 분석 → 시각화 전체 흐름 학습
- SIEM 구성요소 및 동작 원리 이해
- 보안 로그 기반 이상 징후 탐지 경험

## 🛠 기술 스택
- Backend: Spring Boot, JPA, Spring Security, JWT
- Analyzer: Python (로그 파싱 / 통계 / 이상 탐지)
- Frontend: React, TailwindCSS
- DB / Infra: MySQL, AWS

## ✨ 주요 기능
- 보안 로그 업로드 및 저장
- 로그 검색 및 필터링
- 공격 패턴 통계 및 시각화
- JWT 기반 인증 / 역할 기반 접근제어
